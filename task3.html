Task 3: AWS + Vercel Showcase

Objective:
Create AWS infrastructure using Terraform or AWS Console and showcase results via a webpage hosted on Vercel.

Tools:
- Terraform or AWS Console
- Vercel (for web deployment)
- Google Forms (for submission)

Part 1: AWS Resource Setup
1. Amazon EC2
   - Launch a t2.micro instance (Amazon Linux 2)

2. Amazon VPC
   - Create a custom VPC with 1 public subnet + Internet Gateway

3. AWS Lambda
   - Deploy a basic "Hello World" function in Python

4. Amazon S3
   - Private bucket with versioning enabled

5. Amazon EBS
   - Attach 10GB EBS volume to EC2 instance

6. Amazon RDS
   - Launch MySQL db.t2.micro instance

7. Amazon Athena
   - Create a database/table from sample S3 data

8. Amazon CloudWatch
   - Alarm for EC2 CPU Utilization > 70%

9. AWS CloudTrail
   - Enable to monitor API activity

10. AWS CloudFormation
    - Deploy basic template to create EC2

11. AWS CDK
    - Provision an S3 bucket (TypeScript or Python)
